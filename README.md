# gym-project

## Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

## 相关技术
  - 框架：[eggjs](https://eggjs.org/zh-cn/intro/quickstart.html)
  - 数据库：mysql
  - 数据库ORM: [Sequelize](https://itbilu.com/nodejs/npm/VkYIaRPz-.html#induction)
  - 对象存储：七牛云
  - 登录状态：基于session，将扩展token登录
